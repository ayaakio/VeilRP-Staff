1.0 TX Admin

1.1 Opening the Menu

  • Typing /tx opens the TXAdmin menu.
  • Use Tab to move between the different pages of the TX menu.
  • Make sure you understand what page you are on before using any option.

1.2 Main Page

  • The main page contains the most commonly used admin tools.
  • Player Modes include:
    • No Clip
    • God Mode
    • Super Jump
    • Normal
  • Teleport options include:
    • Waypoint
    • Coords
    • Back to Original Spot
    • Copy Coords
  • Vehicle options include:
    • Spawn
    • Fix
    • Delete
    • Boost
  • Heal options include:
    • Myself
    • Everyone
  • You are also able to:
    • Send Announcement
    • Reset World Area (Do Not Touch)
    • Toggle Player IDs
  • Toggling player IDs places IDs above players’ heads along with their CFX name.

1.3 Players Tab

  • The Players tab shows all players currently online.
  • It displays their in-game ID and CFX name.
  • Clicking a player opens a list of actions and information.

1.4 Player Actions

  • Moderation actions include:
    • DM
    • Warn
    • Kick
    • Give Admin
  • Interaction actions include:
    • Heal
    • Go To
    • Bring
    • Spectate
    • Toggle Freeze
  • Troll actions include:
    • Make Drunk
    • Set Fire
    • Wild Attack
  • Do not abuse troll actions. These are not to be used for fun or personal entertainment.

1.5 Player Information

  • TX Admin allows staff to see details about a player, including:
    • Session Time
    • Play Time
    • When They Joined
    • Any Bans or Warns
  • This should be used to gather context before making moderation decisions.

1.6 Player IDs / Identifiers

  • Staff can view a player’s identifiers, including:
    • License
    • Discord
    • FiveM
  • The Discord identifier can be reverse searched to help find the player’s Discord name if required.
  • Use identifiers carefully and only when needed for moderation or investigation.

1.7 Banning

  • TX Admin allows bans to be issued directly through the menu.
  • When banning a player, staff should:
    • Set a clear reason
    • Choose the correct time frame
    • Make sure the punishment is justified
  • Do not issue bans without proper review.

1.8 Server Logs

  • One of the most important sections staff will use is Server Logs.
  • Logs can be searched between dates.
  • Filters include:
    • Player Joins
    • Player Leaves
    • Chat Messages
    • Player Death
    • Menu Actions
    • Explosions
    • Commands
    • System Events
  • These logs should be used to support investigations, reports, and moderation reviews.

1.9 Players Menu on Main TXAdmin

  • Outside the in-game TX menu, staff will also use the main TXAdmin Players menu.
  • This allows players to be searched whether they are online or offline.
  • It provides similar interactions to the in-game Players tab, but with more flexibility and movement options.
  • Use this section when you need more detailed moderation control or account review.

2.0 919 Admin Panel

2.1 Dashboard

  • The Dashboard provides a general overview of players and server activity.
  • It may also include simple command or quick-action abilities depending on permissions.
  • This is useful as a starting point when checking the state of the server.

2.2 Admin Chat

  • Admin Chat is used for staff communication.
  • Use it to discuss staff matters, ongoing situations, or internal support conversations.
  • Keep communication clear and professional.

2.3 Players

  • The Players section shows online players, their character names, and which staff are on duty.
  • From here, staff may be able to give:
    • Jobs
    • Gangs
    • Money
    • Items
  • These tools must only be used with proper reason and never for personal or unjustified benefit.

2.4 Maps

  • The Maps section shows players on the map.
  • This can be useful for finding players, checking activity, or assisting in support situations.
  • Do not use this to interfere in roleplay scenes without reason.

2.5 Reports

  • The Reports section shows any in-game reports placed for issues such as bugs or player concerns.
  • Use this area to review, track, and respond to reports properly.
  • Always gather context before taking action.

2.6 Jobs

  • The Jobs section shows the list of jobs in the database.
  • This is useful when checking if a job exists, confirming job setup, or assisting with staff actions involving employment.

2.7 Gangs

  • The Gangs section shows the list of gangs in the database.
  • Use this when confirming gang setup, checking gang data, or handling staff matters linked to factions.

2.8 Resources

  • The Resources section shows the list of resources in the database.
  • Do not change, stop, or interfere with resources unless you are specifically told to do so and it is part of your role.

2.9 Bans

  • The Bans section shows active bans issued through the 919 menu.
  • Use this to review punishment history, check active bans, and confirm enforcement actions.

2.10 Characters

  • The Characters section lists all characters created on the server.
  • This can be used to review a player’s character list and gather additional context where necessary.

2.11 Server Metrics

  • Server Metrics provides an overall list of server-related information and statistics.
  • Use this to monitor server state and identify anything unusual that may need attention.

2.12 Log Viewer

  • The Log Viewer shows all admin actions taken through the 919 Admin panel.
  • This should be used to review moderation history and confirm staff actions when necessary.

2.13 Items List

  • The Items List shows all items on the server.
  • It may also allow staff to give items to themselves or other players and set a quantity.
  • This must never be used for personal gain.
  • Item tools should only be used for valid staff support reasons.

2.14 Vehicles List

  • The Vehicles List shows all vehicles on the server, including category and price.
  • This is useful for checking vehicle data and confirming setup information when handling player or support questions.

3.0 Best Practice

3.1 Good Staff Practice

  • Always log actions when needed.
  • Never use powers for personal gain.
  • Avoid changing things you cannot justify.
  • Use the minimum level of intervention needed to resolve an issue.
  • Stay neutral, calm, and professional.

4.0 Forbidden Use

4.1 Actions That Are Not Allowed

  • No spawning for yourself.
  • No interfering in personal scenes.
  • No using admin tools to benefit your own character.
  • No punishments without proper review.
  • No misuse of staff tools for fun, convenience, or advantage.
