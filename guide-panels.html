1.0 TX Admin

1.1 Using the In-Game Menu

  • Use the TX Admin in-game menu only when there is a valid staff reason to do so.
  • Make sure you understand what tool you are using before selecting it.
  • Use the menu to assist with staff duties, not to make situations easier for yourself or friends.
  • If you are unsure whether a function is appropriate to use, ask a higher staff member before acting.

1.2 Player Actions

  • Use player actions only when they are needed for moderation, support, or scene management.
  • Always make sure there is a justifiable reason before teleporting, reviving, freezing, spectating, or otherwise affecting a player.
  • Be professional when interacting with players through staff tools.
  • Do not use player actions casually or for entertainment.

1.3 Safe Usage Rules

  • Use TX Admin carefully and only within the limits of your role.
  • Do not touch the resources section.
  • Do not restart, stop, or alter anything you are not explicitly expected to manage.
  • If a situation involves something technical or sensitive, escalate it rather than guessing.

2.0 919 Panel

2.1 Opening the Panel

  • Open the 919 Panel only when needed for a staff action or review.
  • Make sure you are using the correct panel option for the situation before acting.
  • If you are unsure what a function does, do not use it until clarified by senior staff.

2.2 Self Actions

  • Self actions should only be used when necessary for staff duties.
  • Do not use self tools to gain personal advantage in roleplay or gameplay.
  • Any use of self tools must be justifiable if questioned by senior staff or owners.

2.3 Player Actions

  • Use player actions only for staff assistance, moderation, or scene support.
  • Do not interfere with player roleplay unless there is a valid reason.
  • Keep staff interactions fair, neutral, and professional at all times.

2.4 Reports / Punishments

  • Review reports properly before taking action.
  • Make sure punishments match the situation and are supported by evidence.
  • Do not issue punishments based on assumption, frustration, or incomplete context.
  • Escalate situations if they are serious, unclear, or involve conflict of interest.

2.5 Vehicle / Entity Tools

  • Use vehicle and entity tools only when needed to resolve an issue or support staff duties.
  • Do not spawn, remove, or alter vehicles or entities without a clear reason.
  • Any action taken with these tools should be explainable if reviewed later.

2.6 Safe Usage Rules

  • Only use panel options that are relevant to the situation you are handling.
  • Do not experiment with tools on live players or active scenes.
  • If you cannot justify an action clearly, do not do it.
  • Escalate edge cases instead of guessing.

3.0 Best Practice

3.1 Good Staff Practice

  • Always log actions when needed.
  • Never use powers for personal gain.
  • Avoid changing things you cannot justify.
  • Use the least intrusive tool needed to resolve the situation.
  • Stay calm, neutral, and professional in all staff situations.

4.0 Forbidden Use

4.1 Actions That Are Not Allowed

  • No spawning for yourself.
  • No interfering in personal scenes.
  • No using admin tools to benefit your own character.
  • No punishments without proper review.
  • No misuse of staff tools for fun, convenience, or advantage.
