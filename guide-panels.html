1.0 TX Admin

1.1 Opening the Menu

Typing /tx opens the TXAdmin menu.

Use Tab to move between pages.

Make sure you understand what page you are on before using anything.

1.2 Main Page

The main page contains the most commonly used admin tools.

Player Modes include:

  • No Clip
  • God Mode
  • Super Jump
  • Normal

Teleport options include:

  • Waypoint
  • Coords
  • Back to Original Spot
  • Copy Coords

Vehicle options include:

  • Spawn
  • Fix
  • Delete
  • Boost

Heal options include:

  • Myself
  • Everyone

You are also able to:

  • Send Announcement
  • Reset World Area (Do Not Touch)
  • Toggle Player IDs

Toggling IDs shows player IDs above heads with their CFX name.

1.3 Players Tab

Shows all online players with ID and CFX name.

Clicking a player opens actions and information.

1.4 Player Actions

Moderation:

  • DM
  • Warn
  • Kick
  • Give Admin

Interaction:

  • Heal
  • Go To
  • Bring
  • Spectate
  • Toggle Freeze

Troll (Do NOT abuse):

  • Make Drunk
  • Set Fire
  • Wild Attack

1.5 Player Information

Used to gather context before moderation.

  • Session Time
  • Play Time
  • When They Joined
  • Bans / Warns

1.6 Identifiers

Staff can view identifiers including:

  • License
  • Discord
  • FiveM

Discord IDs can be reverse searched if needed.

1.7 Logs

Server logs are heavily used for investigations.

  • Joins / Leaves
  • Chat
  • Deaths
  • Explosions
  • Commands
  • System Events

2.0 919 Admin Panel

2.1 Core Sections

  • Dashboard – overview of server
  • Admin Chat – staff communication
  • Players – manage characters
  • Maps – view players
  • Reports – review issues
  • Jobs / Gangs – database info
  • Resources – DO NOT TOUCH unless told
  • Bans – active bans
  • Characters – all created characters
  • Server Metrics – server data
  • Log Viewer – admin actions
  • Items List – item database
  • Vehicles List – vehicle database

2.2 Important Notes

  • Do not spawn items for personal use
  • Only use tools when required
  • Always be able to justify actions

3.0 Best Practice

  • Log actions when needed
  • No personal gain
  • Avoid unnecessary changes

4.0 Forbidden Use

  • No spawning for yourself
  • No interfering in scenes
  • No abusing admin tools
  • No punishments without review
